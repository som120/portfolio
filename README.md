# somnathpaul.com

My personal portfolio website.

Built with [Next.js](https://nextjs.org/), [shadcn/ui](https://ui.shadcn.com/), [Magic UI](https://magicui.design/), and [Tailwind CSS](https://tailwindcss.com/). Deployed on [Vercel](https://vercel.com).

## Running Locally

```bash
git clone https://github.com/som120/portfolio.git
cd portfolio
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
